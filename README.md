# Sales Data Dashboard - Power BI

Project Overview:

This project is a Sales Data Analysis Dashboard created using Microsoft Power BI.
The dashboard helps to analyze sales performance, revenue trends, and product category performance using interactive visualizations.

Tools Used:

* Microsoft Power BI
* Power Query
* Microsoft Excel

Key Features:

* Sales performance analysis
* Monthly and yearly sales trends
* Product category comparison
* Region-wise sales analysis
* Interactive filters and slicers for better insights

Dashboard Insights:

The dashboard allows users to:

* Identify top-performing products
* Monitor sales growth over time
* Compare sales across different regions
* Make data-driven business decisions
